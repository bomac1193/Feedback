# Geomancy: the deeper root

Snapshot date: 2026-05-09. The frame that ties Feedback, PERI, and the surveyor lineage into one body of work.

## The word

Geomancy. *Geo* (earth) + *manteia* (divination). Earth-divination via patterns drawn in dust, sand, cornmeal, ash. The 5,000-year tradition of reading topology in the ground.

*Geometry* shares the same Greek root: *geo* + *metria* (measurement). Geomancy and geometry split into mystical and rationalist branches. Both encode space as pattern. Both produce permanent records that outlive the person who drew them.

This frame is the missing root for everything else in this network.

## What counts as geomancy

A working list of practices in the same family, all of them earth-pattern coding systems:

| System | Region | Form | Function |
|---|---|---|---|
| **Lusona** | Chokwe (Angola, Zambia, DRC) | Eulerian path drawings in dust, single continuous line, must close | Pedagogy, divination, cosmological teaching |
| **Veve** | Vodou (Haiti, West Africa) | Cosmograms drawn in cornmeal/flour on ground | Invocation; encode which loa can flow which way |
| **Ifa odu** | Yoruba (Nigeria, Benin) | 256 four-line binary figures cast via palm nuts | Divination, recorded patterns of fate |
| **Sikidy** | Madagascar | 16 four-line geomantic figures in tableau grid | Divination via seed-casting, traveled along Indian Ocean trade routes |
| **Khatt al-raml** | Arabic Islamic world | "Sand writing": same 16 figures, drawn in dust | Divination, deeply influenced medieval European geomancy |
| **Western geomancy** | Medieval Europe | Latin-translated khatt al-raml, 16 figures | Court divination (John Dee); now esoteric tradition |
| **I Ching** | Chinese (cousin tradition) | 64 hexagrams cast via yarrow stalks | Divination; same binary-figure logic as ifa |
| **Surveying** | Worldwide rationalist | Coordinates, traverses, datum points, plans | Mapping land for ownership/transit; the technical descendant |

All of them: humans encoding spatial relationships as figure, in materials that mark the earth, in formats that can be transmitted across generations.

## How the work already participates

This wasn't planned, it was inherited.

**Feedback's veve topologies are geomantic.** The instrument has been calling its coupling matrices "veve adjacency" without naming the deeper category. Each veve preset is a geomantic figure governing which oscillator couples to which, at what strength. The Kuramoto network is a geomancy machine. Sound emerges from topology, exactly as veves were designed to do in their original ritual context.

**PERI's surveyor lineage threads back through geomancy.** The user's father was a surveyor — the rationalist branch of the same tradition. Rationalist surveying inherits geomantic earth-reading without naming the inheritance. Coordinates, traverses, datum points: ifa odu translated into Cartesian space. The son extending the father's profession into Gaussian Splatting is the surveyor branch reaching back toward its mystical source.

**The Feedback prompt engine already used Eglash vocabulary.** Eulerian paths (Chokwe), recursive nesting (Ba-ila), branching (Banyo), spirals (Akan). All of these are geomantic primitives. The vocabulary was right; the deeper category name was missing.

## What this changes

### Feedback prompt engine

A fourth media pool added: `MEDIA_GEOMANTIC`. Activates at high Kuramoto sync (r > 0.85) when topology becomes legible as figure. Concrete subjects: ifa odu in cornmeal, sikidy tableau in red sand, khatt al-raml on slate, lusona in white chalk, veve in cornmeal, four-line figures self-organizing in ferrofluid, geomantic structure in bismuth.

The selection logic: when the music locks, the visual becomes explicit divination figure. When it flows or fragments, the visual stays in the cymatic/physical register. The instrument now visually testifies to the geomantic moment when it happens.

### PERI surveyor lineage doc

The surveyor frame becomes "rationalist branch of geomancy." This locates the project in a 5,000-year tradition and makes it harder to dismiss as just AI-art. It also gives the user a way to talk about the work that no other AI-and-grief project can claim: the family profession is, at its root, the mystical practice of reading earth as pattern.

### Cross-project coherence

PERI and Feedback are in the same body now. Not just adjacent projects sharing a stack:

- PERI: voice-as-material from his voice corpus. Audio inheritance.
- Feedback: topology-as-music via veve geomancy. Spatial inheritance.
- Visual layer (when LoRA exists): patterns generated through his lineage. Image inheritance.

Three modes of pattern-reading-pattern, each tied to the same root.

## Vocabulary worth using

For prompts, program notes, write-ups:

- **Geomancy** itself: the umbrella word, locates the project in a real tradition
- **Ifa odu** / **Sikidy figures** / **Khatt al-raml**: specific divination systems
- **Lusona** / **Eulerian paths** / **continuous-line drawings**: Chokwe sand-mathematics
- **Veve** / **cosmogram** / **invocation diagram**: Vodun/Yoruba ground drawings
- **Earth-reading** / **earth-coding** / **pattern divination**: general descriptive terms
- **Rationalist branch** / **mystical branch** of geomancy: when distinguishing surveying from divination
- **Topology as figure** / **figure as oracle**: the conceptual bridge

For any prompt-engine work, keep the "no tribal/ethnic/abstract-art" rule from Feedback's existing philosophy. Geomancy is a real practice, not aesthetic vocabulary. Use specific system names (ifa, sikidy, lusona, khatt al-raml), not generic markers.

## What not to do

- **Don't conflate geomancy with generic "African mysticism."** It's a precise body of practices with regional variants and formal structures. Naming systems specifically (ifa odu, sikidy, lusona) honors their distinctness.
- **Don't use it as decoration in titles.** "Geomantic" as a vibe word will read as appropriation. The work has to actually use the topology, the way Feedback already uses veve adjacency matrices.
- **Don't treat surveying as separate from this lineage.** That's the move that lets the user own the inheritance. Surveying is geomancy continued, just under a different name.

## Where this leads

If the geomantic frame holds — and right now it does — there are at least three clean directions:

1. **A piece titled and structured around a specific odu or veve**, where the geomantic figure governs both the audio coupling (Feedback) and the visual generation pool. The title is the figure's name (e.g., "Eji Ogbe" or "Damballah Wedo").

2. **A methodology piece**, eventually: *Geomancy as instrument architecture: from sand drawings to coupled oscillators to Gaussian Splat scenes.* Locates the work in academic discourse around Eglash, Verran, Saidiya Hartman, et al., without leaning on identity vocabulary.

3. **A geomancy LoRA**, trained on real ifa odu photographs, sikidy tableaux, lusona drawings, veve in cornmeal, khatt al-raml on slate. Use as style anchor for the visual layer alongside the father-art LoRA. The instrument inherits two visual lineages: the personal (father) and the disciplinary (geomancy).
